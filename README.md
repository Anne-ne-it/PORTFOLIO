Mi Portfolio Personal
¡Hola! Bienvenida/o a mi rincón en GitHub. Este repositorio contiene el código fuente de mi **Portfolio Web**, un espacio donde muestro mi trayectoria profesional, mis habilidades en diseño y desarrollo, y mis proyectos más recientes.

Sobre el Proyecto
El objetivo de este sitio es ofrecer una experiencia visual limpia y profesional, destacando mi enfoque en el Diseño UI/UX.

Características principales:
· Diseño Responsive: Adaptado para una lectura en móviles, tablets y escritorio.
· Efectos Visuales: Incluye detalles cuidados como fotos estilo Polaroid y animaciones suaves.
· Menú Navegación sencilla e intuitiva.
· Optimización: Uso de una paleta de colores coherente y tipografía Roboto para una legibilidad máxima.

Tecnologías utilizadas
· HTML5: Estructura semántica del contenido.
· CSS3: Estilizado, variables de entorno (`:root`), Flexbox y Media Queries.
· Git & GitHub: Control de versiones y despliegue.
· Google Fonts: Tipografías optimizadas para web.

Estructura del Código
· index.html: Estructura principal de la página de inicio.
· styles.css: Hoja de estilos centralizada con una organización modular.
